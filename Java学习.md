# Java学习



## 基础语法

### 字面量：

字面量（Literal）是指在源代码中直接表示固定值的语法元素，无需计算或引用变量。例如，在Java中：

- 整数字面量：如 42、0b1010（二进制）、0xFF（十六进制）、1_000_000（带下划线）。
- 浮点字面量：如 3.14、1.23e4（科学计数法）。
- 字符字面量：如 'A'。（单引号）
- 布尔字面量：如 true 或 false。
- 字符串字面量：如 "Hello, literals"。（双引号）
- null字面量：如 null。



```
System.out.println(1_000_000);         // 输出带下划线的整数字面量（用于提高可读性）
```

在Java中，下划线（_）用于数字字面量中分隔数字，提高可读性，但编译时会被忽略。因此，1_000_000 等价于 1000000，输出即为该整数值。
